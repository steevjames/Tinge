HOW TO PLAY


  • RGB stands for RED, BLUE, GREEN.
  • RGB(0,0,0) means amount of Red is 0, Green is 0, Blue is 0.
  • This means RGB(0,0,0,) is black
    (because there are no colors)
  • Maximum Value for a Particular Color is 255.
  • RGB(255,255,255) is White
  • RGB(255,0,0) is Red   
  • RGB(0,255,0) is Green
  • RGB(0,0,255) is Blue  
  • Any Color can be Obtained by Mixing Red, Green and Blue. This notation is very common so its a must know !
